1. Image Acquisition: From Camera to Digital Image

The objective of image acquisition is to generate digital images from sensed data.

  • Analog to Digital: Sensors in cameras capture the projected continuous image in the form of an analog signal (a continuous voltage waveform).
  • The amplitude and spatial behavior of this waveform are related to the physical phenomenon being sensed (e.g., light).
  • To be processed by a computer, this continuous data must be converted into a digital format. This is done via an Analog-to-Digital Converter (ADC), which utilizes two main techniques: Sampling and Quantization.

🎯 EXAM TRIVIA: The Bayer Pattern & Demosaicing

(Extracted from the diagram on Slide 5)

  • Sensor "photosites" only measure brightness, not color.
  • To get color, a Colour Filter Array is placed over the sensor.
  • The filters are arranged in a mosaic known as a Bayer Pattern.
  • Key Fact: The Bayer pattern uses twice as many green filters as blue or red filters to produce a sharper image (because the human eye is more sensitive to green).
  • The process that turns this raw mosaic data into a full-color grid of pixels is called Demosaicing.

2. The Core Definitions: Sampling & Quantization

An image may be continuous with respect to its x- and y-coordinates, and also in its amplitude. To digitize it, we must convert both into discrete values.

Concept Definition Domain
Sampling Digitizing the coordinate values (x, y). Spatial Domain (Time/Space) -> Time-Discrete Signal
Quantization Digitizing the amplitude values (intensity/function). Intensity Domain -> Amplitude-Discrete Signal

The Process flow: Continuous Time/Amplitude ➔ [ SAMPLER ] ➔ [ QUANTIZER ] ➔ Discrete Time/Amplitude (Digital Signal).

3. The Math of Quantization & Gray Level Resolution

In quantization, vertically ranging continuous values are partitioned into discrete levels.

  • Number of Quantas (Partitions) = Number of Gray Levels
  • "Number of gray levels" means the number of different shades of gray available.
  • To improve image quality, we scale the number of gray levels (or "gray level resolution") up.
L = 2k

Where:

  • L = Gray level resolution (Total number of discrete intensity levels, e.g., 256).
  • k = Number of bits per pixel (BPP) / gray level.

According to the slides, these terms are used interchangeably:

Gray level = Number of bits per pixel (BPP) = Number of levels per pixel

Note: If we increase this level to 256 (where k=8), it is known as a standard grayscale image.

4. Resolution: Spatial vs. Intensity

Both types of resolution are digitization-dependent, meaning they depend directly on how the image was sampled and quantized.

Feature Spatial Resolution Intensity Resolution
Definition The size of the smallest perceptible details in an image. The smallest discernible change in the intensity level.
Dependent On Number of Samples / Sampling Rate (N). Number of Bits used for quantization (k).
Measured By Pixels per unit distance (e.g., DPI - Dots Per Inch). Bits per pixel (bpp).
Artifact if too Low Results in Jagged Lines (pixelation/blockiness). Results in False Contouring (banding/visible jumps between shades).
Highest Sensitivity More sensitive to shape variations. More sensitive to lighting variations.
