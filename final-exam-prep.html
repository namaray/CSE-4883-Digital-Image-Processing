🚨 Exam Overview

The final exam heavily emphasizes application and problem-solving over pure memorization. You will be given real-world scenarios (e.g., smart retail, satellite imagery, medical MRIs) and asked to choose and justify the correct DIP techniques. You must also be comfortable solving matrix-based mathematical problems for distance, paths, and convolution.

1. Image Segmentation Scenarios

You must know exactly when to use which type of segmentation. The exam will give you a scenario and ask you to identify the required approach.

Semantic vs. Instance vs. Panoptic

Scenario Correct Approach Why?
Labeling pixels as road, water, or building (Satellite Image) Semantic Segmentation Individual instances don't matter; only the class category is needed.
Counting overlapping cars in a parking lot Instance Segmentation Need to separate and count individual objects of the same class.
Differentiating materials (plastic vs glass) AND counting broken bottles Panoptic Segmentation Requires both background context (Semantic) and individual object separation (Instance).

Traditional vs. Deep Learning Segmentation

🎯 Exam Question Pattern: Justifying DL over Traditional

If asked to segment tumors in MRIs vs. agricultural fields in satellites:

  • Satellite Fields (Traditional): Use region-based or K-means clustering because regions have homogeneous textures, smooth boundaries, and minor boundary imprecision is acceptable.
  • Medical MRI Tumors (Deep Learning/CNN): Traditional methods fail because tumor boundaries are irregular, intensity distributions overlap with healthy tissue, and there is high inter-patient variability. An Encoder-Decoder CNN (like SegNet or U-Net) is required to learn complex, non-linear representations.

2. CNN & Deep Learning Concepts

Classification vs. Object Detection

Know the difference in output and application:

  • Image Classification: Answers "Is the shelf full?" or "Is the image healthy?". Output layer is a Softmax (class probabilities).
  • Object Detection: Answers "Where is the Coca-Cola bottle?" or "Where are the pests?". Output layer contains bounding box coordinates (bx, by, bw, bh) + class c + confidence score.

CNN Layers & Mathematics

  • Convolution Math: Be prepared to manually calculate the output of a convolution operation given a small matrix, a kernel, a specified stride, and padding (zero-padding or copy-padding).
  • Activation Functions: Why use ReLU after convolution? It introduces non-linearity, allows the network to learn complex patterns, is computationally efficient, and helps mitigate the vanishing gradient problem.
  • Pooling Strategies:
    ➔ Use Max Pooling for extracting sharp edges, corners, and highly discriminative features (e.g., symbols).
    ➔ Use Average Pooling for assessing overall texture, intensity distribution, and surface uniformity.

3. Morphological Operations (Visual Identification)

You will be given "Before" and "After" images (matrices or visual shapes) and asked to identify the operation.

🎯 Identification Guide:

  • If the object shrinks, loses boundary pixels, or thin protrusions disappear ➔ Erosion
  • If the object grows, expands, or bridges gaps between nearby objects ➔ Dilation
  • If small noise outside the object disappears, but the main object stays roughly the same size ➔ Opening (Erosion followed by Dilation)
  • If small holes inside the object or gaps are filled, but the main object size stays the same ➔ Closing (Dilation followed by Erosion)

Always be ready to "Briefly explain its working" (e.g., "Closing works by first dilating the image to fill gaps, then eroding to restore the original boundary size without recreating the filled gaps").

4. Path, Connectivity & Distance Matrices

This is a guaranteed numerical/matrix problem. You must know how to trace paths and calculate distances by hand.

Paths & Connectivity

  • 4-Path / 8-Path: Can you reach pixel B from pixel A stepping only on 1s? For 4-path, you can only move Up/Down/Left/Right. For 8-path, diagonals are allowed.
  • m-Path: Allowed to move to an 8-neighbor ONLY IF there is no shared 4-neighbor that is also a 1. Tracing an m-path is a common question (e.g., "Find the m-path distance Dm between (1,2) and (5,4)").

Distance Measures

🎯 Matrix Calculation Skills:

You may be given a matrix and asked to "Find the city-block distance of all pixels from (2,3)". You must populate the matrix with the correct distances.

  • Euclidean (De): √((x1-x2)² + (y1-y2)²) — Straight line.
  • City-Block (D4): |x1-x2| + |y1-y2| — Diamond shape contour.
  • Chessboard (D8): max(|x1-x2|, |y1-y2|) — Square shape contour.

Trick Question: "Highlight common pixels covered by contours of x and y that satisfy City-block distance <= 3". You must mentally draw the diamond contour radius 3 around both points and find the intersection!
