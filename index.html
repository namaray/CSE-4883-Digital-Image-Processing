8. Morphological Image Processing ✓ Notes Ready

  • Structuring Element (SE), FIT, HIT, MISS concepts
  • Erosion (A ⊖ B): Shrinks objects, FIT condition
  • Dilation (A ⊕ B): Grows objects, HIT condition
  • Opening = Erode then Dilate (removes protrusions)
  • Closing = Dilate then Erode (fills holes/gaps)
  • Exam: Robot gap-filling → use Closing

9. Image Segmentation ✓ Notes Ready

  • Types: Semantic, Instance, Panoptic Segmentation
  • Traditional: Thresholding, Region-Based (Watershed), Edge Detection (Canny)
  • Clustering-Based: K-means Segmentation
  • Deep Learning: Encoder-Decoder Architecture (SegNet)
  • Applications: Robotics, Medical Imaging, Autonomous Vehicles, Satellites

10. CNN for Image Classification & Object Detection ✓ Notes Ready

  • Object Detection: Bounding Boxes (bx, by, bw, bh, c)
  • YOLO Algorithm: Grid-based real-time detection
