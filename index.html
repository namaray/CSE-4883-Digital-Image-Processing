🎓 Final Syllabus

Topics from the final portion of the course — fully documented below.

5. Image Enhancement in Spatial Domain

  • Point Processing (Thresholding, Negative, Log Transformations)
  • Histogram Processing & Equalization
  • Spatial Filtering (Smoothing, Sharpening, Laplacians)

6. Image Enhancement in Frequency Domain

  • Fourier Transforms
  • Low-pass & High-pass Filters

7. Morphological Image Processing

  • Structuring Element, FIT, HIT, MISS
  • Erosion (A ⊖ B): Shrinks objects, FIT condition
  • Dilation (A ⊕ B): Grows objects, HIT condition
  • Opening = Erode then Dilate (removes protrusions)
  • Closing = Dilate then Erode (fills holes/gaps)
  • Exam: Robot gap-filling → use Closing

8. Image Segmentation

  • Types: Semantic, Instance, Panoptic Segmentation
  • Traditional: Thresholding, Region-Based (Watershed), Edge Detection (Canny)
  • Clustering-Based: K-means Segmentation
  • Deep Learning: Encoder-Decoder Architecture (SegNet)
  • Applications: Robotics, Medical Imaging, Autonomous Vehicles, Satellites

9. CNN for Image Classification & Object Detection

  • Images as Matrices; Kernels & Feature Maps
  • CNN Architecture: Conv, Stride, Padding, Pooling, Batch Norm
  • Image Classification Pipeline (Cat vs. Dog)
  • Object Detection: Bounding Boxes (bx, by, bw, bh, c)
  • YOLO Algorithm: Grid-based real-time detection
  • Exam: Classifier vs. Detector; Pest detection with bounding boxes
