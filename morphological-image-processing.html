1. What is Morphological Image Processing?

Morphological Image Processing (MIP) is a collection of techniques based on image shape. Operations probe an image with a small shape called a structuring element (SE), and based on how that shape fits into the image, pixels are classified.

Why use MIP?

  • Remove imperfections in segmented images (noise, gaps, stray pixels).
  • Deliver information on shape and structure of objects in an image.
  • Used for boundary detection, region filling, and skeletonization.

🎯 Key Terminology

  • Structuring Element (SE): A small matrix/template used to traverse an image — similar to a convolution kernel, but works differently. It defines the probe shape.
  • FIT: The SE perfectly fits inside the image region — every SE '1' overlaps with an image '1'.
  • HIT: At least one SE '1' overlaps with an image '1' (partial match).
  • MISS: No SE '1' overlaps with any image '1' (complete mismatch).

2. Basic Morphological Operations

🔻 Erosion (A ⊖ B)

Effect: Shrinks objects by removing boundary pixels. Eliminates small noise objects and thin protrusions.

Rule:

  • Output pixel = 1 if SE FITS (all SE 1s match image 1s)
  • Output pixel = 0 otherwise

🔼 Dilation (A ⊕ B)

Effect: Grows objects by adding boundary pixels. Fills holes and gaps, connects broken parts.

Rule:

  • Output pixel = 1 if SE HITS (at least one SE 1 matches an image 1)
  • Output pixel = 0 otherwise

Use Cases

  • Erosion: Separating lightly-touching objects, removing thin noise lines, extracting boundaries (original minus eroded).
  • Dilation: Filling small holes, connecting broken object parts, thickening thin structures.

3. Compound Morphological Operations

Compound operations combine Erosion and Dilation to achieve results neither can produce alone. There are two key compound operations:

📂 Opening (A ∘ B)

Steps: Erosion FIRST, then Dilation

A ∘ B = (A ⊖ B) ⊕ B

Effect: Removes small foreground protrusions and tiny objects, without significantly changing the size of larger objects.

Use: Cleaning noise while preserving main shapes.

📁 Closing (A • B)

Steps: Dilation FIRST, then Erosion

A • B = (A ⊕ B) ⊖ B

Effect: Fills small holes and gaps within foreground objects without significantly altering overall size.

Use: Filling discontinuities in broken outlines.

⚠️ EXAM TRAP: Order of Operations

  • Opening = Erode FIRST, then Dilate → Removes small protrusions.
  • Closing = Dilate FIRST, then Erode → Fills small gaps/holes.

Memory tip: "Closing Closes gaps" — Dilation closes the gap before Erosion restores size.

✅ Final Exam Application (Spring 2025, Q3a)

A binary robot image has gaps at joints, claws, and limbs. Which MIP technique should be used to complete the image without changing its overall shape?

Answer: Closing (A • B) — Dilation fills the gaps first; Erosion then restores the overall size. This is precisely what Closing is designed for.
