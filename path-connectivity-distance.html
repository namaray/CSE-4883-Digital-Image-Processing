1. Digital Path

A digital path (or digital curve) from pixel p with coordinates (x, y) to pixel q with coordinates (s, t) is a sequence of distinct pixels with coordinates:

(x₀, y₀), (x₁, y₁), ..., (xₙ, yₙ)

Where:

  • (x₀, y₀) = (x, y) — the starting pixel p
  • (xₙ, yₙ) = (s, t) — the ending pixel q
  • Each consecutive pair (xᵢ, yᵢ) and (xᵢ₋₁, yᵢ₋₁) must be adjacent for 1 ≤ i ≤ n
  • n is the length of the path
  • If (x₀, y₀) = (xₙ, yₙ) — the path is closed

🎯 Types of Paths

  • 4-path: Consecutive pixels must be 4-adjacent (only horizontal/vertical moves).
  • 8-path: Consecutive pixels must be 8-adjacent (horizontal, vertical, or diagonal moves).
  • m-path: Uses m-adjacency rules to eliminate ambiguities in diagonal connections.

2. Connectivity

Let S be a subset of pixels in an image.

  • Two pixels p and q are connected in S if there exists a path between them consisting entirely of pixels in S.
  • For any pixel p in S, the set of all pixels connected to it in S is called a connected component of S.

Regions and Boundaries

  • Region R: A subset of pixels in an image where R is a connected set.
  • Boundary (Border/Contour): The set of pixels in region R that have one or more neighbors that are not in R.
  • Special case: If R is the entire image, its boundary is defined as the pixels in the first and last rows and columns.

Foreground and Background

Suppose an image contains K disjoint regions Rᵢ (none touching the image border):

  • R₹: The union of all K regions.
  • Foreground: R₹ (the union of all regions).
  • Background: (R₹)c — the complement of R₹.

3. Distance Measures

A distance function D between pixels p(x,y), q(s,t), and z(v,w) must satisfy three properties:

  1. D(p, q) ≥ 0, and D(p, q) = 0 if and only if p = q
  2. D(p, q) = D(q, p) (symmetry)
  3. D(p, z) ≤ D(p, q) + D(q, z) (triangle inequality)

Three Distance Metrics

DistanceNameFormulaAlso Called
De Euclidean Distance De(p,q) = √[(x-s)² + (y-t)²] Straight-line distance
D4 City-Block Distance D4(p,q) = |x-s| + |y-t| Manhattan distance, Taxicab distance
D8 Chessboard Distance D8(p,q) = max(|x-s|, |y-t|) Chebyshev distance

🎯 Worked Example: p=(2,2), q=(1,1)

  • Euclidean: De = √[(2-1)² + (2-1)²] = √2 ≈ 1.414
  • City-Block (D4): |2-1| + |2-1| = 1 + 1 = 2
  • Chessboard (D8): max(|2-1|, |2-1|) = max(1, 1) = 1 (because q is a direct 8-neighbor)

🎯 Using D4 to Prove 4-Neighbors

For a pixel p at (2,2), all pixels with D4 = 1 are its 4-neighbors:

  • Pixel A at (2,1): |2-2| + |1-2| = 0 + 1 = 1 ✓
  • Pixel B at (3,2): |3-2| + |2-2| = 1 + 0 = 1 ✓
  • Pixel C at (2,3): |2-2| + |2-3| = 0 + 1 = 1 ✓
  • Pixel D at (1,2): |1-2| + |2-2| = 1 + 0 = 1 ✓

Pixels with D4 ≤ 2 from (x,y) form diamond-shaped contours of constant distance.

Pixels with D8 ≤ 2 from (x,y) form square-shaped contours of constant distance. (Final Exam Q3c: D8 ≤ 2 from (3,2) forms a 5×5 square centered at that pixel.)

4. Dm (Mixed/m-Path Distance)

The Dm distance is defined as the length of the shortest m-path between two points. Unlike D4 and D8, Dm depends on the pixel values along the path and their neighbors (specifically, V = the set of allowed pixel values).

Case Analysis for Dm

Consider pixels p, p1, p2, p3, p4 with p, p2, p4 = 1 and p1, p3 = variable. V = {1}.

⚠️ Why Does Dm Vary?

The m-path distance changes because m-adjacency eliminates diagonal ambiguity. When a common 4-neighbor of two diagonal pixels is in V, the diagonal connection is broken and the path must go around, increasing the distance.
