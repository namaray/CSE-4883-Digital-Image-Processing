What is a Digital Image?

An image is represented as a two-dimensional function f(x,y):

f(x,y)
  • x and y are the spatial (plane) coordinates.
  • The amplitude of f at any pair of coordinates (x,y) is called the intensity or gray level of the image at that point.

Digital Image Definition: If x, y, and the amplitude values of f are all finite and discrete quantities, we call the image a digital image.

It is composed of a finite number of elements called pixels (picture elements), each of which has a particular location and value.

Types of Digital Images

Images are classified based on the number of bits used to represent the intensity of each pixel:

Image Type Description
Binary Image Contains only two pixel elements: 0 (Black) and 1 (White). Also known as a Monochrome or Bitmap image.
Black & White Image Consists of strictly black and white colors, with no intermediate grays.
8-bit Color Format The most common grayscale format. It has 256 different shades of gray. 0 stands for Black, 255 stands for White, and 127 is medium gray.
16-bit Color Format Also called High Color Format. Can display up to 65,536 colors. Great for saving memory but limiting for professional graphics.
24-bit Color Format Also called True Color. Using 24 bits, it can display 16,777,215 colors. It distributes 8 bits for Red (R), 8 bits for Green (G), and 8 bits for Blue (B).
32-bit Color Format Like 24-bit color, but features an extra 8-bit Alpha Channel for gradients, shadows, and transparency. Supports over 4.2 billion combinations.

Image as a Matrix

In computers, images are represented as matrices of numbers organized in rows and columns. This is the syntax for a digital image of size M × N:

f(x,y) =
[ f(0,0), f(0,1), ... f(0,N-1) ]
[ f(1,0), f(1,1), ... f(1,N-1) ]
[ ... , ... , ... ... ]
[ f(M-1,0), f(M-1,1), ... f(M-1,N-1) ]

Breakdown:

  • The matrix is composed of M rows and N columns.
  • Every element in this matrix is a picture element or pixel.
  • In an 8-bit image, every position in the matrix holds a value ranging from 0 to 255.

How a Computer Sees Images

While we see smooth colors and shapes, the computer processes grids of digital numbers:

  • Grayscale: 1 set of digits per pixel (e.g., 11111111 representing 255).
  • RGB Image: 3 sets of digits per pixel (8 bits for Red, 8 bits for Green, and 8 bits for Blue).

Tip: You can inspect pixel grid numeric values directly on websites like pixspy.com.
