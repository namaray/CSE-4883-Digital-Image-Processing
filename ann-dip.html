1. Forms of Learning in AI

An agent is "learning" if it improves its performance on future tasks after making observations about the world. There are three main types based on feedback:

  • Unsupervised Learning: Agent learns patterns in the input with NO explicit feedback supplied. (Example: Clustering "good traffic days" vs "bad traffic days").
  • Reinforcement Learning: Agent learns from a series of reinforcements (rewards or punishments). (Example: Winning points in chess).
  • Supervised Learning: Agent observes example input-output pairs and learns a function that maps inputs to outputs.

🎯 EXAM TRIVIA: Supervised Learning Categories

  • Classification: Output y is a finite set of values (e.g., sunny, cloudy, rainy). If there are only two values (0 or 1), it is called Binary Classification.
  • Regression: Output y is a continuous number (e.g., tomorrow's temperature).

Definitions to know:
Training Set: N example input-output pairs.
Hypothesis: A possible function for generating output.
Test Set: Examples distinct from the training set.

2. Basics of Artificial Neural Networks (ANN)

ANNs are brain-inspired systems intended to replicate how humans learn. They consist of nodes (imitating biological neurons) connected by links (imitating axons/dendrites).

Topology / Architecture

  • Single-layer perceptron: Suitable for simple, linear problems.
  • Multi-layer perceptron: Suitable for complex, non-linear problems.
  • FeedForward vs Feedback: In FeedForward, data moves only in one direction (left to right).

Layers of an ANN

  1. Input Layer: Number of neurons corresponds to the number of features/inputs.
  2. Hidden Layer(s): Arbitrary number of layers with an arbitrary number of neurons. This is where input transformations occur.
  3. Output Layer: Number of neurons corresponds to the number of expected output values.
Mathematical Model of a Node: a = g(W*X + b)

Where X = Inputs, W = Weights, b = Bias, and g() = Activation Function. The result a is called the activation or node value.

3. Activation Functions (Transfer Functions)

Their main purpose is to convert an input signal of a node into an output signal. They map the resulting values to specific ranges.

Function Type Range Characteristics / Exam Notes
Linear Linear (-∞ to +∞) Equation: f(x) = x. Output is not confined. Rarely used in complex hidden layers.
Sigmoid (Logistic) Non-Linear (0 to 1) S-shaped. Crucial for probability prediction. Best for Binary Classification output layers.
Tanh Non-Linear (-1 to 1) Also S-shaped but crosses zero. Generally better than Sigmoid for hidden layers in feed-forward nets.
ReLU Non-Linear [0 to +∞) Rectified Linear Unit. Most used activation function in the world right now (especially in CNNs and Deep Learning). Half-rectified: f(z) = 0 if z < 0, and f(z) = z if z >= 0.

3.5 Mathematical Simulation of a Forward Pass

Based on the architecture slides, here is a step-by-step matrix calculation showing how data moves through an ANN and how different activation functions alter the data at each layer.

🧠 The Setup

We have an ANN with 3 inputs, 2 hidden layers, and 1 output layer.

  • Inputs (I): [0.1, 0.3, 0.5]
  • Expected Outputs (y): [0.0, 1.0, 0.0]
  • Biases (b): Assumed to be [1.0, 1.0, 1.0] for each layer.
  • Note: Weights (W) are randomly initialized to small numbers between 0 and 1.

Step 1: Layer 1 Matrix Operation + ReLU Activation

First, we calculate the dot product of Inputs and Weights, then add the bias.

[h1_in] = [Inputs] × [Weights_L1] + [Biases]

[0.1, 0.3, 0.5] × W_ij + [1.0, 1.0, 1.0] = [1.2, 1.6, 1.5]

Next, we apply the ReLU Activation Function: ReLU = max(0, x)

  • max(0, 1.2) = 1.2
  • max(0, 1.6) = 1.6
  • max(0, 1.5) = 1.5

Layer 1 Output: [1.2, 1.6, 1.5]

Step 2: Layer 2 Matrix Operation + Sigmoid Activation

We take the output from Layer 1 and multiply it by the weights of Layer 2.

[h2_in] = [h1_out] × [Weights_L2] + [Biases]

[1.2, 1.6, 1.5] × W_jk + [1.0, 1.0, 1.0] = [3.0, 3.0, 2.4]

Next, we apply the Sigmoid Activation Function: Sigmoid = 1 / (1 + e-x)

  • 1 / (1 + e-3.0) = 0.952
  • 1 / (1 + e-3.0) = 0.952
  • 1 / (1 + e-2.4) = 0.916

Layer 2 Output: [0.952, 0.952, 0.916]

Step 3: Layer 3 Matrix Operation + Softmax Activation

We take the output from Layer 2 and multiply it by the weights of Layer 3.

[o_in] = [h2_out] × [Weights_L3] + [Biases]

[0.952, 0.952, 0.916] × W_kl + [1.0, 1.0, 1.0] = [1.7, 2.9, 2.7]

Because this is the final layer and we want a probability distribution that sums to 1 across multiple classes, we apply the Softmax Activation Function: Softmax = ex / Σ ex

After calculating the exponential proportions for all three nodes, we get:

Final Predicted Output (o_out): [0.1420, 0.4717, 0.3862]

📉 Step 4: Measuring the Generated Error

We have finished the forward pass. Now we compare our prediction to the actual truth.

  • Expected Output: [0.0, 1.0, 0.0]
  • Our Prediction: [0.1420, 0.4717, 0.3862]

The network uses the Logistic Loss (Log Loss) / Binary Crossentropy formula to calculate exactly how wrong this prediction is, which is then used in Back-Propagation to fix the weights:

Log Loss = -yi * log(o_outi) - (1 - yi) * log(1 - o_outi)

4. ANN Training Methodology (The 7 Steps)

How an ANN actually learns. Memorize this sequence for the exam:

  1. Initialize: Randomly initialize the Weights to a small number close to 0 (but not exactly 0).
  2. Input: Input the first observation of your dataset into the input layer (one feature per node).
  3. Forward-Propagation: From left to right, neurons are activated. Propagate the activations until you get the predicted result y.
  4. Error Evaluation: Compare the predicted result to the actual expected result to measure the generated error.
  5. Back-Propagation: From right to left, the error is back-propagated. Update the weights according to how much they are responsible for the error. The Learning Rate decides how drastically to update the weights.
  6. Repeat: Repeat steps 1 to 5, updating weights after each observation.
  7. Epoch: When the whole training set has passed through the ANN, that constitutes one epoch. Redo for multiple epochs.

⚖️ Gradient Descent & Batch Sizes

The update algorithm used to improve the model (move down the error gradient). The slide explicitly defines these three variations:

  • Batch Gradient Descent: Batch Size = Size of Entire Training Set.
  • Stochastic Gradient Descent (SGD): Batch Size = 1.
  • Mini-Batch Gradient Descent: 1 < Batch Size < Size of Training Set.

5. Practical Implementation (Keras Workflow)

Based on the Bank Customer Churn dataset (predicting if a customer leaves or stays).

A. Data Preprocessing

  • Libraries: numpy (arrays), pandas (datasets), matplotlib (graphs).
  • Encoding Categorical Data: Converting text (Geography, Gender) into numbers using LabelEncoder and OneHotEncoder.
  • Splitting: train_test_split (usually 80% train, 20% test).
  • Feature Scaling: Crucial to ensure high numeric values (like Salary) don't overpower low ones (like Age). Use StandardScaler.

B. Building the Model

# Importing Keras
from keras.models import Sequential # Initializes ANN model
from keras.layers import Dense      # Adds layers to ANN

# 1. Initialize
classifier = Sequential()

# 2. Add Input & First Hidden Layer
# units=6 (nodes in hidden layer), activation='relu', input_dim=11 (11 features in dataset)
classifier.add(Dense(units=6, activation='relu', input_dim=11))

# 3. Add Second Hidden Layer (input_dim not needed anymore)
classifier.add(Dense(units=6, activation='relu'))

# 4. Add Output Layer
# units=1 (binary output: leave/stay), activation='sigmoid'
classifier.add(Dense(units=1, activation='sigmoid'))

C. Compiling & Training

# Compile the Model
classifier.compile(optimizer='adam', loss='binary_crossentropy', metrics=['accuracy'])

# Train the Model
classifier.fit(X_train, y_train, batch_size=10, epochs=100)

Exam Notes on Compilation:

  • Optimizer: Algorithms to change weights/learning rate to reduce loss. Examples: Adam, SGD, Adagrad, RMSprop.
  • Loss Function: We use binary_crossentropy (or Log Loss) because the target attribute is binary (0 or 1).
  • Metrics: Evaluates performance. Examples: accuracy, F1 score.

D. Prediction & Evaluation

# Predict results (Returns probabilities due to Sigmoid)
y_pred = classifier.predict(X_test)

# Convert probability to Binary (0 or 1) based on 0.5 threshold
y_pred = (y_pred > 0.5)

# Evaluate using Confusion Matrix
from sklearn.metrics import confusion_matrix
cm = confusion_matrix(y_true=y_test, y_pred=y_pred)
